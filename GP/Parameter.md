

# World Parameter

- guardAlert : 외부 침입 내성
- merchantTrust : 찾아오는 상인 우호도
- enemyAggressive : 적의 공격성 + 약탈자의 위협
- shelterComfort : hub 분위기
- radiation : 밖 오염도

# Expedition State

- expeditionTurn : 탐험 지난 턴 수
- expeditionMoveCount : 움직임 수
- optionalKillCount : 평균 킬보다 오버 킬한 정도
- avoidCount : 싸움 회피형 움직임 정도
- farmingCount : 파밍 횟수
- talkCount : npc나 적과의 대화 횟수

# Player Profile

- mercy : 자비
- greedy : 욕심
- curious : 호기심
- discipline : 엄격, 규율
- risk : 위험성 있는
- social : 사회적
- cruel : 잔인함
- caution : 조심성
